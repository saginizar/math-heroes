# MathHeroes / גיבורי המספרים

A local browser-based math learning game for kids. Superhero-themed, Hebrew-native, fully offline PWA.

## How to Play

1. Serve locally: `python -m http.server 8080`
2. Open on tablet: `http://<your-pc-ip>:8080`
3. Tap "Add to Home Screen" for PWA install
4. Play offline forever

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript (zero dependencies)
- Web Speech API (Hebrew TTS)
- Web Audio API (procedural SFX)
- LocalStorage (save/progress)
- PWA (offline-capable)
